PROJECTS

DAILY JOURNAL

This Daily Journal project was created to practice using MongoDB and the mongoose framework in particular. User can compose a new post and view all the other posts in the Home page. All the post are rendered by querying data stored in the database. (P.S) Currently all the posts will be lost once the user closes the window or restarts the server.

TO-DO LIST v2

To-Do List v2, is slightly different to other lists as the user can have multiple to-do lists. By adding a new name at the end of the url, a new list will be created with the given name as the title. Users can then add items to the lists and delete the item after completion by clicking the checkbox. All the saved lists and their respective items are stored within the MongoDB database. The list can be created, updated and deleted by the user, and each action will update the data stored in the database.

GURKHA ONE

An iOS based application built using React Native, Node.js (Express.js) that displays football tournaments held annually by the Nepalese community. Users can search for a specifc team and add them or remove them from their 'favourites' list. This project was used to get an idea of develping a mobile application and practice React Native components along with a bit of TypeScript.
